# Code of Conduct

## Our Commitment

We are committed to making participation in Baton welcoming, respectful, and harassment-free for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while giving clear feedback.
- Focus on ideas, not people.
- Accept and incorporate valid criticism.
- Help keep discussions on-topic and useful.

## Unacceptable Behavior

- Harassment, threats, or personal attacks
- Discriminatory language or behavior
- Trolling, baiting, or deliberate disruption
- Publishing others' private information without permission
- Any conduct that would be inappropriate in a professional community

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.

Maintainers may remove, edit, or reject comments, commits, issues, and other contributions that violate this policy.

## Reporting

If you experience or witness unacceptable behavior:

1. Open a private GitHub security advisory in this repository if the issue is sensitive.
2. If private advisory is not suitable, open an issue marked `conduct` with minimal necessary detail.

## Enforcement Guidelines

Maintainers may take actions including:

- Warning
- Temporary ban from project spaces
- Permanent ban from project spaces

Decisions will be based on severity, pattern of behavior, and community safety.
